# sortingByFrequency

## Project requirements
- Node.js version 12.13, 14.15, 16.10 and above.
- Vue CLI 4.x
## Project setup
### Install Vue CLI
```
npm install -g @vue/cli
# OR
yarn global add @vue/cli
```
### Install packages
```
npm run installAll
```

### Build frontend
``` 
npm run build
```
### Start local server
```
npm run start
```
